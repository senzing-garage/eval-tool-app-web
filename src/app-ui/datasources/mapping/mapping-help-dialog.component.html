How to Map Unmapped Files

Use the Senzing MCP server with Claude Code or Claude Desktop to automatically map your source data files (CSV, JSON) into Senzing-ready format.

  1. Set up the Senzing MCP server

    Add the Senzing MCP server to your Claude Code or Claude Desktop configuration. In Claude Code, run claude mcp add to register the server. In Claude Desktop, add it under Settings > MCP Servers.

  2. Prepare your source file

    Have your source data file ready as CSV, JSON, or JSONL.

  3. Ask Claude to map your file

    In Claude Code or Claude Desktop, ask:

    map '/path/to/your-file.csv' to senzing

    The MCP mapping workflow will profile your data, plan the entity types, map fields, generate a mapper script, and validate the output.

  4. Run the generated mapper script

    Execute the Python mapper script that Claude generated to produce a .jsonl output file containing Senzing-formatted records.

  5. Import the mapped file

    Use the Add Data Source button on this page to select and import the mapped .jsonl file.
